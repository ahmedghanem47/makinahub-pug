# Workflow

This is a simple workflow for building HTML/CSS/JavaScript applications based on Gulp. It uses:
* Pug as an HTML preprocessor
* PostCSS
* Babel for JavaScript
* Browserify for bundling

## How to use

### Starting development server

```sh
# YARN
yarn start

# NPM
npm run start
```

### Creating a build

```sh
# YARN
yarn build

# NPM
npm run build
```

### Serving a build

```sh
# YARN
yarn serve

# NPM
npm run serve
```
